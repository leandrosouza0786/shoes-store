## Technologies

- Angular 9+
- SASS

## How to Use

To run this project you'll need GIT, NodeJS and Angular installed on your computer.

```bash
# clone project
$ git clone https://github.com/leandrosouza0786/shoes-store

# Go into the repository
$ cd shoes-store

# Install dependencies
$ npm install

# Run the project
$ ng serve

# Run JSON server
$ npm run api

```
